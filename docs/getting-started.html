# 快速上手

# 环境

使用 Vack 需依赖以下环境：

  • Node.js v16.14.2+open in new window
  • pnpmopen in new window（可选）

# 脚手架

Vack 提供了一个项目创建工具 —— create-vack 。

  1. 使用 create-vack 快速创建 Vack 项目：
pnpm create vack

npm create vack@latest

yarn create vack
  1. 跟随指引输入项目名称并指定预设：
✔ 项目名称： … vack-project
✔ 是否需要接口Mock： … yes
✔ 是否需要多语言： … yes
✔ 是否需要eslint： … yes
✔ 提交代码时检查代码格式： … yes
✔ 是否规范commit提交信息： … yes

项目创建完成，cd vack-project 后可以：

    安装依赖：npm install
    开发调试：npm run dev
  1. 也可以通过命令行直接指定项目名称、预设
pnpm create vack my-app --preset mock,i18n

npm create vack@latest my-app --preset mock,i18n

npm create vack@latest my-app -- --preset mock,i18n

yarn create vack my-app --preset mock,i18n

当前支持以下预设：

预设说明
mock带有接口模拟能力
i18n带有国际化能力
eslint带有 ESLint 配置
lint-staged提交时要求代码通过规范校验
lint-commit提交时要求commit信息通过规范校验
